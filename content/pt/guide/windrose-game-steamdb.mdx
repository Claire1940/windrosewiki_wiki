## Configurando seu Servidor com SteamCMD

Antes de começar, certifique-se de que seu hardware atende aos padrões de 2026 para hospedagem de jogos de sobrevivência. Você precisará de um ambiente baseado em Windows (embora o suporte ao Linux seja frequentemente gerenciado via Proton/Wine) e uma conexão de internet estável com a capacidade de encaminhar portas no seu roteador.

### Passo 1: Inicialize o SteamCMD
Baixe o utilitário SteamCMD na Valve Developer Community. Extraia os arquivos para uma pasta dedicada em sua unidade, como `C:\SteamCMD`. Execute o executável para permitir que ele atualize e construa a estrutura de arquivos necessária.

### Passo 2: Buscar os Arquivos do Servidor
Ao chegar ao prompt de comando, siga estas etapas para baixar o servidor usando o App ID do **windrose game steamdb**:

1. Digite `login anonymous` e pressione Enter. (Windrose permite login anônimo, portanto você não precisa usar suas credenciais pessoais da conta Steam).
2. Digite `app_update 4129620` e pressione Enter.
3. Aguarde a mensagem "Success! App '4129620' fully installed".
4. Digite `exit` para fechar o prompt.

> **⚠️ Aviso:** Abrir portas no seu roteador pode ser arriscado. Certifique-se de que o Firewall do Windows esteja configurado para permitir apenas o tráfego nas portas específicas exigidas pelo jogo para evitar acesso não autorizado à sua rede.

## Configuração e Personalização

Após baixar os arquivos, navegue até a pasta do SteamCMD e depois para `steamapps\common\Windrose Dedicated Server`. Este diretório contém o executável e os arquivos de configuração necessários para personalizar sua experiência pirata. 

### Modificando a Descrição do Servidor
A primeira vez que você executar o arquivo `StartServerForeground.bat`, o jogo gerará uma pasta `Saved`. Dentro desta pasta, você encontrará um diretório `Config` contendo o `serverDescription.json`. É aqui que você define a identidade do seu servidor.

| Parâmetro | Propósito | Valor de Exemplo |
|---|---|---|
| **ServerName** | O nome exibido na lista de servidores | **"Golden Age Pirate Cove"** |
| **MaxPlayerCount** | Máximo de jogadores simultâneos | **16** |
| **IsPasswordProtected** | Ativar/Desativar entrada de senha | **true** |
| **Password** | A string necessária para entrar | **"arrgh2026"** |
| **InviteCode** | Código único para conexão direta | **Gerado automaticamente** |

### Configurações Avançadas
Para aqueles que desejam manipular ainda mais o mundo do jogo — como ajustar as taxas de ressurgimento de recursos ou a dificuldade de combate — o jogo requer um segundo arquivo de configuração. Embora nem sempre esteja presente por padrão, o diretório do servidor geralmente inclui um arquivo `DedicatedServer_MD` que descreve os requisitos para configurações JSON avançadas. Você pode criar um JSON personalizado para substituir os comportamentos padrão do mundo, permitindo uma experiência de sobrevivência sob medida.

## Solução de Problemas Comuns

Mesmo com o ID correto do **windrose game steamdb**, você pode encontrar obstáculos durante o processo de configuração. Em 2026, os problemas mais comuns giram em torno da verificação de arquivos e erros de credenciais.

*   **Erro App State 0x...:** Isso geralmente indica um problema de conexão com os servidores do Steam. Tente executar o comando `app_update` novamente.
*   **Executável Ausente:** Certifique-se de que está procurando na pasta `common`. Às vezes, o SteamCMD instala arquivos em um subdiretório diferente se o caminho de instalação não for especificado.
*   **Lag de Combate:** Se os jogadores sentirem "rubber-banding" durante o combate naval, considere aumentar a prioridade de CPU do processo do servidor no Gerenciador de Tarefas do Windows.
*   **Falha no Login do SteamCMD:** Embora o `login anonymous` funcione para o Windrose, certifique-se de que sua internet não esteja bloqueando a API do Steam.

> **💡 Dica:** Para automatizar atualizações, crie um arquivo `.bat` que execute o SteamCMD com os comandos `+login anonymous +app_update 4129620 validate +quit`. Execute isso sempre que um novo patch do jogo for lançado.

**P: Como atualizo meu servidor Windrose quando um novo patch é lançado?**

R: Você deve executar o SteamCMD novamente, fazer login anonimamente e executar o comando `app_update 4129620`. Isso baixará os patches mais recentes e os aplicará aos arquivos de servidor existentes.

**P: Posso rodar o jogo e o servidor dedicado no mesmo PC?**

R: Embora seja possível, não é recomendado. Rodar ambos exige recursos significativos de RAM e CPU, o que pode causar lag tanto para o anfitrião quanto para os jogadores conectados. É melhor usar um PC secundário ou um serviço de hospedagem especializado.

**P: Onde encontro o Código de Convite para meus amigos?**

R: O Código de Convite está localizado no arquivo `serverDescription.json`, que é encontrado no diretório `Saved\Config` da sua pasta de servidor após a primeira inicialização bem-sucedida.
