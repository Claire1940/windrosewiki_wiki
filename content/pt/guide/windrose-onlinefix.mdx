## Instalação Passo a Passo do windrose onlinefix

O processo de instalação do **windrose onlinefix** é simples, mas requer precisão em relação aos diretórios das pastas. Você está essencialmente substituindo as DLLs de rede padrão por versões modificadas que redirecionam o tráfego através do wrapper Steamworks.

1.  **Baixar os Arquivos do Fix**: Obtenha o arquivo compactado contendo os arquivos do fix rotulados especificamente para a versão 0.10.0.3.104.256f9653.
2.  **Extrair Conteúdo**: Use uma ferramenta como WinRAR ou 7-Zip para extrair os arquivos. Você normalmente verá um `SteamworksFix.ini` e vários arquivos `.dll`.
3.  **Localizar o Diretório do Jogo**: Navegue até a pasta onde o Windrose está instalado. Geralmente encontrado em `SteamLibrary\steamapps\common\Windrose` ou em um caminho de instalação personalizado.
4.  **Substituir Arquivos**: Copie os arquivos extraídos e cole-os no diretório raiz do jogo. Quando solicitado, selecione "Substituir arquivos no destino".
5.  **Iniciar Steam**: Certifique-se de estar logado em uma conta Steam válida. Frequentemente recomenda-se usar uma conta secundária se você estiver preocupado com a privacidade da sua biblioteca.
6.  **Executar o Executável**: Inicie o jogo através do `Windrose.exe` (ou o lançador específico fornecido com o fix).

## Mecânicas de Multiplayer e Hospedagem

Uma vez que o **windrose onlinefix** seja aplicado com sucesso, o menu do jogo exibirá novas opções para conectividade multiplayer. Ao contrário do modo offline padrão, você verá agora uma aba "Social" ou "Lobby".

### Hospedando uma Sessão
Para hospedar, selecione "Criar Expedição" no menu principal. Você pode configurar o lobby como "Apenas Amigos" ou "Público". Como você está usando o **windrose onlinefix**, seus amigos verão você jogando "Spacewar" ou outro título temporário em sua lista de amigos da Steam. Eles podem clicar com o botão direito no seu nome e selecionar "Participar do Jogo" para entrar diretamente na sua sessão.

### Entrando em uma Sessão
Existem duas formas principais de entrar:
*   **Convite Direto**: O host envia um convite através do overlay da Steam (Shift+Tab).
*   **Navegador de Servidores**: Navegue até o menu multiplayer e atualize a lista. Lobbies criados usando a mesma versão do fix aparecerão aqui.

| Recurso | Capacidades do Host | Capacidades do Convidado |
| :--- | :--- | :--- |
| **Persistência do Mundo** | Salva na unidade local do Host | O progresso sincroniza com o Host |
| **Limite de Jogadores** | Até 8 jogadores (v0.10) | Limitado pela largura de banda do Host |
| **Trocas** | Gerenciamento total de inventário | Pode trocar itens com o Host |
| **Combate** | Define o escalonamento de dificuldade | Participa de encontros em grupo |

## Solução de Problemas Comuns do windrose onlinefix

Mesmo com uma instalação perfeita, problemas técnicos podem ocorrer devido a configurações de firewall ou conflitos de software. Se o **windrose onlinefix** não parecer estar funcionando, o primeiro passo é verificar se o overlay da Steam está ativo. Se o overlay não aparecer quando você pressionar Shift+Tab, o fix não foi inicializado corretamente.

### Conexão Expirada (Connection Timed Out)
Este é o erro mais comum em 2026. Geralmente indica uma incompatibilidade de versão. Certifique-se de que todos os jogadores do grupo estejam usando exatamente a mesma build do jogo e do **windrose onlinefix**. Mesmo uma pequena diferença de patch pode impedir o aperto de mão (handshake) entre os clientes.

### Erro "No Licenses"
Se a Steam exibir um erro de "No Licenses", certifique-se de ter adicionado o aplicativo temporário (como o Spacewar) à sua conta. Você pode fazer isso pressionando `Win+R` e digitando `steam://install/480`. Você não precisa baixar o jogo completo; apenas ter a licença registrada em sua conta costuma ser suficiente para o fix funcionar.

> **💡 Dica:** Se você estiver enfrentando alta latência, tente desativar qualquer VPN ativa. Embora as VPNs às vezes ajudem com problemas de NAT, elas frequentemente interferem no tunelamento P2P usado pelo fix online.

## FAQ

**P: O windrose onlinefix é seguro para usar na minha conta principal da Steam?**

R: Embora muitos jogadores usem fixes online sem problemas, sempre há um risco teórico ao modificar arquivos de jogo que interagem com a API da Steam. A maioria dos usuários prefere usar uma conta secundária ("alt") para jogar jogos modificados para garantir que sua biblioteca principal não seja afetada por possíveis violações de serviço.

**P: Por que meus amigos não conseguem ver o lobby que eu hospedei?**

R: Isso geralmente se deve a uma de três coisas: um firewall bloqueando o executável do jogo, a Steam não estar aberta ou uma incompatibilidade de versão. Certifique-se de que os arquivos do **windrose onlinefix** foram copiados para a pasta do jogo de todos e que todos estão executando a mesma versão (0.10.0.3.104.256f9653).

**P: Este fix permite o cross-play entre diferentes plataformas?**

R: Não, o atual **windrose onlinefix** foi projetado especificamente para jogadores de PC baseados em Windows que usam a API Steamworks. O cross-play com consoles ou outras lojas não é suportado através desta modificação específica.

**P: Preciso reinstalar o fix após uma atualização do jogo?**

R: Sim. Se o jogo receber uma atualização oficial, ela provavelmente substituirá os arquivos DLL modificados. Você precisará esperar que uma versão compatível do fix seja lançada e então reaplicá-la aos novos arquivos do jogo.
