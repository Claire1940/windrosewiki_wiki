## Einrichten Ihres Servers mit SteamCMD

Bevor Sie beginnen, stellen Sie sicher, dass Ihre Hardware den Standards von 2026 für das Hosting von Survival-Spielen entspricht. Sie benötigen eine Windows-basierte Umgebung (obwohl Linux-Support oft über Proton/Wine realisiert wird) und eine stabile Internetverbindung mit der Möglichkeit, Ports an Ihrem Router weiterzuleiten.

### Schritt 1: SteamCMD initialisieren
Laden Sie das SteamCMD-Tool von der Valve Developer Community herunter. Entpacken Sie die Dateien in einen dedizierten Ordner auf Ihrem Laufwerk, z. B. `C:\SteamCMD`. Führen Sie die ausführbare Datei aus, damit sie sich aktualisieren und die notwendige Dateistruktur aufbauen kann.

### Schritt 2: Server-Dateien abrufen
Sobald Sie die Eingabeaufforderung erreicht haben, folgen Sie diesen Schritten, um den Server unter Verwendung der **windrose game steamdb** App-ID herunterzuladen:

1. Geben Sie `login anonymous` ein und drücken Sie die Eingabetaste. (Windrose erlaubt einen anonymen Login, sodass Sie nicht Ihre persönlichen Steam-Zugangsdaten verwenden müssen).
2. Geben Sie `app_update 4129620` ein und drücken Sie die Eingabetaste.
3. Warten Sie auf die Meldung "Success! App '4129620' fully installed".
4. Geben Sie `exit` ein, um die Eingabeaufforderung zu schließen.

> **⚠️ Warnung:** Das Öffnen von Ports an Ihrem Router kann riskant sein. Stellen Sie sicher, dass Ihre Windows-Firewall so konfiguriert ist, dass sie nur Datenverkehr auf den spezifischen, vom Spiel benötigten Ports zulässt, um unbefugten Zugriff auf Ihr Netzwerk zu verhindern.

## Konfiguration und Anpassung

Navigieren Sie nach dem Herunterladen der Dateien zu Ihrem SteamCMD-Ordner und dann zu `steamapps\common\Windrose Dedicated Server`. Dieses Verzeichnis enthält die ausführbare Datei und die Konfigurationsdateien, die zur Personalisierung Ihres Piratenerlebnisses erforderlich sind.

### Serverbeschreibung ändern
Wenn Sie die Datei `StartServerForeground.bat` zum ersten Mal ausführen, generiert das Spiel einen Ordner namens `Saved`. In diesem Ordner finden Sie ein `Config`-Verzeichnis, das die Datei `serverDescription.json` enthält. Hier legen Sie die Identität Ihres Servers fest.

| Parameter | Zweck | Beispielwert |
|---|---|---|
| **ServerName** | Der in der Serverliste angezeigte Name | **"Golden Age Pirate Cove"** |
| **MaxPlayerCount** | Maximale Anzahl gleichzeitiger Spieler | **16** |
| **IsPasswordProtected** | Passworteingabe aktivieren/deaktivieren | **true** |
| **Password** | Die zum Beitritt erforderliche Zeichenfolge | **"arrgh2026"** |
| **InviteCode** | Eindeutiger Code für Direktverbindung | **Wird automatisch generiert** |

### Erweiterte Einstellungen
Für diejenigen, die die Spielwelt weiter beeinflussen möchten – etwa durch Anpassung der Ressourcen-Spawnraten oder des Kampfschwierigkeitsgrades – benötigt das Spiel eine zweite Konfigurationsdatei. Obwohl nicht immer standardmäßig vorhanden, enthält das Serververzeichnis oft eine `DedicatedServer_MD`-Datei, die die Anforderungen für fortgeschrittene JSON-Konfigurationen skizziert. Sie können eine benutzerdefinierte JSON-Datei erstellen, um Standard-Weltverhalten zu überschreiben und so ein maßgeschneidertes Survival-Erlebnis zu ermöglichen.

## Verbinden mit Ihrem Windrose-Server

Das Beitreten zu einem dedizierten Server in Windrose unterscheidet sich geringfügig von anderen Survival-Titeln. Anstelle eines traditionellen Server-Browsers, der das gesamte Internet scannt, nutzt Windrose ein **Einladungscode-System**.

1. **Windrose starten:** Öffnen Sie das Spiel über Steam.
2. **Charakterauswahl:** Wählen oder erstellen Sie Ihren Piraten-Avatar.
3. **Verbinden:** Wählen Sie "Connect to a Server" im Hauptmenü.
4. **Code eingeben:** Fügen Sie den Einladungscode ein, den Sie in Ihrer `serverDescription.json`-Datei finden.
5. **Beitreten:** Klicken Sie auf die Schaltfläche zum Beitreten, um die Welt zu betreten.

Wenn Sie den Server für Freunde außerhalb Ihres lokalen Netzwerks hosten, müssen Sie sicherstellen, dass die Ports Ihres Routers weitergeleitet sind. Wenn Ihr Server nicht erscheint oder die Verbindung unterbrochen wird, überprüfen Sie, ob Ihre öffentliche IP erreichbar ist und ob die **windrose game steamdb** Dateien vollständig auf die neueste Version aktualisiert wurden.

## Fehlerbehebung bei häufigen Problemen

Selbst mit der korrekten **windrose game steamdb** ID können während des Einrichtungsprozesses Hürden auftreten. Im Jahr 2026 drehen sich die häufigsten Probleme um die Dateiverifizierung und Anmeldefehler.

*   **App State 0x... Fehler:** Dies deutet normalerweise auf ein Verbindungsproblem mit den Steam-Servern hin. Versuchen Sie, den Befehl `app_update` erneut auszuführen.
*   **Fehlende ausführbare Datei:** Stellen Sie sicher, dass Sie im Ordner `common` suchen. Manchmal installiert SteamCMD Dateien in einem anderen Unterverzeichnis, wenn der Installationspfad nicht angegeben wurde.
*   **Kampf-Verzögerungen (Lag):** Wenn Spieler während des Schiffskampfes "Rubber-Banding" erleben, sollten Sie in Erwägung ziehen, die CPU-Priorität des Serverprozesses im Windows Task-Manager zu erhöhen.
*   **SteamCMD-Login fehlgeschlagen:** Während `login anonymous` für Windrose funktioniert, stellen Sie sicher, dass Ihr Internet die Steam-API nicht blockiert.

> **💡 Tipp:** Um Updates zu automatisieren, erstellen Sie eine `.bat`-Datei, die SteamCMD mit den Befehlen `+login anonymous +app_update 4129620 validate +quit` ausführt. Führen Sie diese immer aus, wenn ein neuer Spiele-Patch veröffentlicht wird.

Die Verwaltung einer Live-Umgebung erfordert Vertrautheit mit der Konsole. Während das Fenster des dedizierten Servers ein Protokoll der Ereignisse anzeigt, können Sie spezifische Befehle verwenden, um die Ordnung aufrechtzuerhalten.

| Befehl | Aktion |

## FAQ

**F: Muss ich das Spiel auf Steam besitzen, um einen dedizierten Server zu hosten?**

A: Nein. Durch die Verwendung von SteamCMD und der **windrose game steamdb** App-ID mit einem anonymen Login können Sie die Serverdateien herunterladen und hosten, ohne eine separate Kopie des Spiels für den Server-Rechner erwerben zu müssen.

**F: Wie aktualisiere ich meinen Windrose-Server, wenn ein neuer Patch veröffentlicht wird?**

A: Sie müssen SteamCMD erneut ausführen, sich anonym anmelden und den Befehl `app_update 4129620` ausführen. Dies lädt die neuesten Patches herunter und wendet sie auf Ihre vorhandenen Serverdateien an.

**F: Kann ich das Spiel und den dedizierten Server auf demselben PC ausführen?**

A: Das ist zwar möglich, wird aber nicht empfohlen. Beides gleichzeitig auszuführen erfordert erhebliche RAM- und CPU-Ressourcen, was zu Verzögerungen sowohl für den Host als auch für die verbundenen Spieler führen kann. Es ist besser, einen zweiten PC oder einen spezialisierten Hosting-Dienst zu nutzen.

**F: Wo finde ich den Einladungscode für meine Freunde?**

A: Der Einladungscode befindet sich in der Datei `serverDescription.json`, die im Verzeichnis `Saved\Config` Ihres Serverordners nach dem ersten erfolgreichen Start zu finden ist.
