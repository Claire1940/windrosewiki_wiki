## Die Mechanik hinter der Windrose-Duplizierung

Um zu verstehen, wie man diese Exploits sicher durchführt, muss man zunächst verstehen, wie die Spielarchitektur funktioniert. Windrose nutzt ein entkoppeltes Speichersystem. Das bedeutet, dass Ihre Charakterdaten – einschließlich Inventar, ausgerüsteter Ausrüstung und persönlichem Lager – in einer völlig separaten Datei von den Weltdaten gespeichert werden. Die Weltdaten enthalten den Zustand der Umgebung, einschließlich Truhen, fallen gelassener Gegenstände und Gebäude.

Da der Charakter und die Welt nicht so miteinander „kommunizieren“, dass ihre Speicherzustände gleichzeitig synchronisiert werden, entsteht ein Schlupfloch. Wenn Sie einen Gegenstand aus einer Truhe in einer bestimmten Welt nehmen, wird dieser Gegenstand Ihrem Charakterspeicher hinzugefügt. Wenn Sie dann den Weltspeicherstand auf einen früheren Zustand zurücksetzen, in dem die Truhe noch voll war, existiert der Gegenstand effektiv an zwei Orten gleichzeitig: in der Tasche Ihres Charakters und in der Truhe der Welt.

| Speichertyp | Gespeicherte Daten | Dateispeicherort-Kategorie |
| :--- | :--- | :--- |
| **Charakterspeicher** | Inventar, Level, Ausrüstung, Schiffslager | SaveProfiles / Player |
| **Weltspeicher** | Truheninhalt, Gebäude, Gelände, NPCs | SaveProfiles / Worlds |

> **💡 Tipp:** Stellen Sie immer sicher, dass Sie einen Ordner „Master-Backup“ auf Ihrem Desktop haben, bevor Sie versuchen, Dateien zu verschieben. Ein falsches Löschen kann zum Verlust von dutzenden Stunden Fortschritt führen.

## Schritt-für-Schritt-Anleitung zur Windrose-Duplizierung

Die Durchführung der **Windrose-Duplizierung** erfordert keine externe „Cheat“-Software oder Trainer von Drittanbietern. Es ist ein manueller Prozess, bei dem lokale Speicherdateien manipuliert werden. Befolgen Sie diese Schritte sorgfältig, um sicherzustellen, dass der Prozess funktioniert, ohne Ihre Daten zu beschädigen.

### 1. Finden Sie Ihr Speicherverzeichnis
Der erste Schritt besteht darin, herauszufinden, wo das Spiel Ihren Fortschritt speichert. Die meisten modernen Windows-Installationen speichern dies im AppData-Ordner.
*   Drücken Sie die **Windows-Taste + R** auf Ihrer Tastatur.
*   Geben Sie `%appdata%` ein und drücken Sie die Eingabetaste.
*   Navigieren Sie einen Ordner zurück zum Verzeichnis **Local**.
*   Suchen Sie den Ordner mit der Bezeichnung **R5**. Dies ist das primäre Verzeichnis für Windrose-Daten.
*   Folgen Sie dem Pfad: `R5 -> Saved -> SaveProfiles`.

### 2. Sichern Sie Ihre Zielwelt
Sobald Sie sich im Ordner `SaveProfiles` befinden, sehen Sie einen Unterordner namens **Worlds**. Darin wird jede von Ihnen erstellte Welt durch eine eindeutige ID-Zeichenfolge dargestellt.
*   Identifizieren Sie die Welt, die die Gegenstände enthält, die Sie duplizieren möchten.
*   Klicken Sie mit der rechten Maustaste auf den Ordner und wählen Sie **Kopieren**.
*   Fügen Sie diesen Ordner an einem temporären Backup-Speicherort ein (z. B. in einen Ordner auf Ihrem Desktop mit dem Namen „Windrose Backups“).

### 3. Entnehmen Sie die Gegenstände
Nachdem Ihr Backup gesichert ist, starten Sie das Spiel.
*   Loggen Sie sich in die Welt ein, die Sie gerade gesichert haben.
*   Verschieben Sie die gewünschten Gegenstände aus den Truhen oder Lagerbehältern der Welt in das Inventar Ihres Charakters oder Ihr Schiffslager.
*   Loggen Sie sich aus der Welt zurück ins Hauptmenü aus. Sie müssen nicht unbedingt den gesamten Spielclient schließen, aber Sie müssen sich im Login-Bildschirm befinden.

### 4. Den Weltzustand wiederherstellen
Da die Gegenstände nun sicher auf Ihrem Charakter gespeichert sind, müssen Sie die Welt „zurücksetzen“, damit die Gegenstände wieder in den Truhen erscheinen.
*   Gehen Sie zurück zu Ihrem **Worlds**-Ordner im Windows-Datei-Explorer.
*   Löschen Sie die aktuelle Version des Welt-Ordners (den, aus dem Sie gerade Gegenstände entnommen haben).
*   Kopieren Sie Ihren Backup-Ordner vom Desktop und fügen Sie ihn wieder in das Verzeichnis `Worlds` ein.
*   Loggen Sie sich wieder in das Spiel ein. Sie werden feststellen, dass sich die Gegenstände immer noch in Ihrem Inventar befinden und auch in den Welttruhen respawnt sind.

## Importieren von All-Item-Speicherständen

Eine beliebte Variante der **Windrose-Duplizierung** besteht darin, „All-Item“-Speicherstände zu verwenden, die von der Community bereitgestellt werden. Dies sind Weltspeicherstände, in denen erfahrene Spieler jede Waffe, jedes Rüstungsset und jedes Material im Spiel in ordentlich beschrifteten Truhen organisiert haben.

Um eine dieser Dateien zu verwenden:
1.  Laden Sie die Weltspeicherdatei herunter (normalerweise ein Ordner mit einem langen ID-Namen).
2.  Schließen Sie Windrose vollständig.
3.  Navigieren Sie zu Ihrem Ordner `Local/R5/Saved/SaveProfiles/Worlds`.
4.  Fügen Sie den heruntergeladenen Ordner in dieses Verzeichnis ein.
5.  Starten Sie das Spiel. Die neue Welt erscheint in Ihrer Weltenauswahl-Liste.

| Item-Kategorie | Bemerkenswerter Loot in geteilten Welten | Nutzen |
| :--- | :--- | :--- |
| **Waffen** | Legendäre Entermesser, verzauberte Bögen | Hoher Schaden für Bosskämpfe |
| **Rüstung** | Schwere Plattenrsets, Stealth-Ausrüstung | Überleben in hochstufigen Zonen |
| **Materialien** | Hartholz, raffinierter Stahl, Stein | Essenziell für **massive Bauprojekte** |
| **Haustiere** | Seltenes Wildschwein-Pet-Item | Kampfunterstützung und Buffs |

## Bauen im großen Stil mit duplizierten Materialien

Der Hauptgrund, warum Spieler die **Windrose-Duplizierung** nutzen, ist für groß angelegte Architektur. Der Bau einer Burg oder einer massiven Schiffsflotte erfordert tausende Einheiten Holz, Stein und Eisen. Unter normalen Spielbedingungen könnte ein Solospieler Wochen damit verbringen, Inseln abzuholzen, nur um eine einzige Festung zu bauen.

Durch das Duplizieren von Stapeln von „Hartholz“ oder „raffiniertem Stein“ können Sie sich auf den kreativen Aspekt des Spiels konzentrieren. Wenn das Lager Ihres Charakters voll mit duplizierten Materialien ist, können Sie Ihre „Hauptwelt“ betreten und ohne Einschränkungen bauen.

> **Warnung:** Während das Duplizieren von Materialien im Allgemeinen sicher für Ihren Speicherstand ist, sollten Sie vermeiden, Truhen über ihre vorgesehene Kapazität hinaus zu füllen, da dies gelegentlich dazu führen kann, dass die Weltdatei nicht geladen wird oder unerwartet auf einen früheren Zustand zurückgesetzt wird.

## Best Practices für die Speicherverwaltung

Um sicherzustellen, dass Ihre Bemühungen zur **Windrose-Duplizierung** nicht zum Verlust des Fortschritts führen, halten Sie sich an diese professionellen Standards für die Dateiverwaltung:

1.  **Temporäre Backups löschen:** Das Spiel generiert manchmal eigene interne Backups. Wenn Sie Ordner mit „.bak“-Endungen oder zusätzlichen Nummern sehen, ist es oft am besten, diese in einen anderen Ordner zu verschieben, um zu verhindern, dass das Spiel „verwirrt“ wird, welcher Weltzustand geladen werden soll.
2.  **Versionskontrolle:** Wenn Sie mehrere Duplizierungsrunden durchführen, beschriften Sie Ihre Backup-Ordner mit Zeitstempeln (z. B. „Welt_Backup_April25_1400“).
3.  **Charaktersicherheit:** Erstellen Sie immer ein separates Backup Ihres „Player“-Ordners innerhalb von `SaveProfiles`. Während der Duplizierungsprozess auf die Weltdateien abzielt, schützt Sie ein Charakter-Backup vor versehentlichem Löschen des Inventars.
4.  **Spiel-Updates:** Wenn die Entwickler im Jahr 2026 einen großen Patch für Windrose veröffentlichen, können Duplizierungsmethoden, die auf Dateimanipulation basieren, manchmal nicht mehr funktionieren. Überprüfen Sie nach einem Patch immer die Community-Foren, bevor Sie versuchen, Dateien auszutauschen.

Weitere Informationen zu offiziellen Spielmechaniken und Updates finden Sie im [Windrose Steam Community Hub](https://steamcommunity.com/app/windrose) für die neuesten Patchnotizen und Einblicke der Entwickler.

## FAQ

**F: Kann ich für die Verwendung der Windrose-Duplizierung gebannt werden?**

A: Da Windrose primär lokale Speicherdateien für seine Welt- und Charakterdaten verwendet, ist die Durchführung von Duplizierungen auf Dateiebene in einer privaten oder Einzelspieler-Umgebung im Allgemeinen nicht durch Anti-Cheat-Systeme nachweisbar. Die Verwendung dieser Methoden auf öffentlichen, von Entwicklern gehosteten Servern kann jedoch gegen deren spezifische Nutzungsbedingungen verstoßen.

**F: Funktioniert diese Methode für das Wildschwein-Pet-Item?**

A: Ja. Jeder Gegenstand, der in einer Truhe platziert oder in einer Welt auf den Boden fallen gelassen werden kann, kann mit der Welt-Wiederherstellungsmethode dupliziert werden. Dies schließt seltene Haustier-Items, legendäre Waffen und einzigartige Questbelohnungen ein.

**F: Warum erscheint die neue Welt nicht, nachdem ich den Ordner eingefügt habe?**

A: Stellen Sie sicher, dass die Ordnerstruktur korrekt ist. Der Ordner, den Sie in `Worlds` einfügen, sollte die `.db`- oder `.save`-Dateien direkt enthalten. Wenn die Dateien in einem zusätzlichen Unterordner verschachtelt sind (z. B. `Worlds/Neuer_Ordner/Neuer_Ordner/Dateien`), wird das Spiel die Welt nicht erkennen.

**F: Gibt es ein Limit dafür, wie viele Gegenstände ich duplizieren kann?**

A: Das einzige Limit ist die Kapazität Ihres Charakter-Inventars und Ihres Schiffslagers. Sie können den Prozess der **Windrose-Duplizierung** so oft wiederholen, wie Sie möchten, um jeden verfügbaren Lagerplatz zu füllen, den Sie besitzen.
