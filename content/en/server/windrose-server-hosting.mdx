A dedicated server handles the heavy lifting of world simulation, enemy AI, and physics calculations, which are particularly taxing during intense naval combat.

## Connecting to Your Server

Joining a dedicated server in Windrose is slightly different from other survival games. Instead of a traditional server browser, the game utilizes the **Invite Code** system for direct connections.

1.  Launch Windrose and create/select your character.
2.  Select **Play** from the main menu.
3.  Choose **Connect to Server**.
4.  Enter the **Invite Code** provided in your hosting dashboard.
5.  Click **Find**, select your server, and enter the password when prompted.

> **Warning:** Ensure your server status is "Running" in the web dashboard before attempting to connect. If the server was recently restarted, it may take 60-90 seconds to become visible in the network.

## Advanced World Customization

One of the greatest benefits of **windrose server hosting** is the ability to modify the `world description.json` file. This allows you to scale difficulty, quest progression, and resource drops beyond the standard 100% limits.

### Accessing the File Manager
To make these changes, your server must be **stopped**. Navigate to the following directory in your provider's File Manager:
`R5/saved/save/profiles/default/rocksDB/0.10/worlds/[YourWorldSeed]/`

Locate the file named `world description.json`. It is highly recommended to download a backup of this file to your local PC before editing.

### Understanding Difficulty Multipliers
In the JSON file, values are represented as decimals. A value of `1.0` equals 100% (default). If you want to make the game significantly harder or easier, use the following logic:

| Setting | JSON Value | In-Game Effect |
| :--- | :--- | :--- |
| **Default Difficulty** | `1.0` | Standard experience |
| **Hardcore Combat** | `2.5` | 250% Enemy Damage/Health |
| **Extreme Challenge** | `5.0` | 500% Difficulty (Maximum) |
| **Relaxed Exploration** | `0.5` | 50% Difficulty (Casual) |

You can also toggle `shared quest progression` within this file. Setting this to `true` ensures that when one player completes an objective, the entire crew receives credit, which is vital for maintaining group cohesion during long voyages.

## Optimizing Server Performance

To keep your **windrose server hosting** running smoothly throughout 2026, consider these maintenance tips:

*   **Scheduled Restarts:** Configure your server to restart automatically once every 24 hours. This clears the RAM cache and resolves minor physics glitches that can accumulate over long sessions.
*   **Automatic Updates:** Enable "Automatic Updates" in your dashboard. Windrose frequently releases hotfixes in Early Access, and players with mismatched versions will be unable to connect.
*   **File Verification:** If you experience frequent crashes, use the "Verify Game Files" tool in your server dashboard to ensure the installation isn't corrupted.

For more information on game mechanics and official patches, visit the [Windrose Steam Community Hub](https://store.steampowered.com/app/1404370/Wind_Rose/) for the latest developer updates.

## FAQ

**Q: How many players can a Windrose server handle?**

A: Currently, **windrose server hosting** supports a maximum of 8 players. However, for the best performance and gameplay balance, a group size of 4 to 6 is recommended by the community and developers.

**Q: Can I change my server's location after purchasing?**

A: Most providers like G-Portal allow you to switch regions, but it may require a server reinstall. It is best to choose the location closest to the majority of your players during the initial setup to avoid high latency.

**Q: What happens if I forget my Invite Code?**

A: Your Invite Code is always visible in your server's "Basic Settings" or "Dashboard" tab. You can change it at any time, but remember that all players will need the new code to reconnect.

**Q: Do I need to keep my PC on for the server to run?**

A: No. The benefit of dedicated **windrose server hosting** is that the game world runs on the provider's hardware. You can turn off your computer, and your friends can continue playing as long as the server is active in the dashboard.
